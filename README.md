# Coin Math

Coin Math - решайте примеры и получайте VK Coin

Бот: [@coinmath](https://vk.me/coinmath)

## Установка

1. Скачайте или клонируйте репозиторий

2. Установите [Python](https://www.python.org/downloads) версии 3.7 или выше

3. Установите необходимые модули:
    ```bash
    pip3 install -r requirements.txt
    ```

4. Установите [PostgreSQL](https://postgresql.org/download/) и создайте базу данных

5. Создайте группу ВКонтакте и получите токен со следующими правами: _управление сообществом, сообщения сообщества, фотографии сообщества_

6. Запустите _main.py_. Это создаст файл конфигурации (config.json), который необходимо заполнить, используя таблицу параметров ниже

7. Сгенерируйте примеры с помощью команды
    ```bash
    main.py --generate 100
    ```

8. Запустите бота (_main.py_)

## Конфигурация

| Параметр       | Описание |
| -------------- | -------- |
| group_domain   | Короткий адрес группы |
| group_id       | ID группы |
| token          | Токен группы |
| vkc_id         | ID страницы, для которой получен ключ API VK Coin |
| vkc_key        | Ключ API VK Coin |
| db_url         | Строка параметров подключения к PostgreSQL |
| levels         | Кол-во опыта, необходимое для достижения уровней и кол-во VK Coin, получаемое за решение примеров |
